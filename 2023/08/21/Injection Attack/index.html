group by (分组)

可用二分法进行列数判断 ==> select * from ysers where id=1 group by 2;

order by语句（排序）

ORDER BY 是SQL的一个子句，用于对查询结果进行排序。

ORDER BY 子句在SQL注入攻击中主要是用于进行盲注和判断数据库中列的数量。

这是因为，如果尝试使用一个比实际列数更大的数字来排序，数据库将返回一个错误

limit语句

limit 是 SQL 中的一个子句，它的作用是限制查询结果返回的记录数

group_concat()函数（合并成一行显示）

GROUP_CONCAT()是一个SQL聚合函数，通常用在MySQL数据库查询中。它会返回一个字符串结果，该字符串是由组（group）中的所有值通过逗号（或其他指定的分隔符）连接起来的。

extractvalue函数

MySQL中的 EXTRACTVALUE() 函数是用于从XML片段中提取值的。

在SQL注入的报错注入中经常使用到该函数

updatexml函数

updatexml(xml_tartget,xpath_expr,new_xml)

xml_tartget 需要操作的xml片段

xpath_expr 需要更新的xml路径(Xpath格式)

new_xml 更新后的内容

​ 3）输入的SQL语句能否不报错—我们的语句能成功闭合
