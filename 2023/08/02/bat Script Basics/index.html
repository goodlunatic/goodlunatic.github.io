字符串截取

使用%var:~n,k%命令，其中%var表示待截取的字符串，~是取字符标志符，n表示起始位置，k表示结束位置(不包含)

字符串替换

使用%var:old_str=new_str%命令

DOS基本命令

rem

rem注释符，也可以用两个冒号代替(::)

echo

显示信息，或启用或关闭命令回显。@字符放在命令前将关闭该命令回显，无论此时echo是否为打开状态

pause

暂停并输出“请按任意键继续. . .”

errorlevel

程序执行结果返回码，执行成功返回0，失败返回1

start

启动一个单独的窗口以运行指定的程序或命令

exit

退出CMD.EXE程序或当前批处理脚本

cls

清屏

help

显示帮助页面
