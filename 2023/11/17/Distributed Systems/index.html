选择题

  1. What is a distribured system?(B)

    A. A system with multiple processors connected by a high-speed network
    B. A system where tasks are divided among multiple interconnected computers
    C. A system with a single mainframe computer
    D. A system where tasks are performed on a single computer only.

  2. What is the role of fault tolerance in distributed systems?(C)
    A. To increase the number of faults in the system.
    B. To prevent any faults from occurring.
    C To detect and recover from faults to ensure system reliability
    D. To ignore faults and continue normal operation.

    1. Which of the following is a challenge in distributed systems related to consistency?(D)
      A. Load balancing
      B. Fault tolerance
      C. Security
      D. Maintaining data consistency across distributed components.

  3. Which archiectural mode is commonly used in web applications where the client is a web browser and the server hosts the web application logic?(A)
    A. Client-Server
    B. Master-Slave
    C. Three-Tier
    D. Peer-to-Peer

  4. What is the primary purpose of networking in a distributed system?(B)
    A. To connect multiple keyboards and monitors.
    B. To enable communication and data exchange between distributed components.
    C. To enhance the processing speed of individual computers
    D. To provide power supply to all connected devices

  5. What does TCP/IP stand for in the context of networking protocols?(B)

    Transmission Control Protocol/Internet Protocol 中文意思为“传输控制协议/网络协议

    A. Technical Communication Protocol/Internet Protocol
    B. Transmission Control Protocol/Internet Protocol
    C. Telecommunication Control Protocol/Internal Protocol
    D. Transfer Control Protocol/Intranet Protocol

  6. What is a distributed file system?(B)
    A. A system where files are stored on a single central server.
    B. A system where files are divided and stored across multiple computers
    C. A system where files can only be accessed locally
    D. A system where files cannot be shared among users
