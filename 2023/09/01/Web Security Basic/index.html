3、User-Agent（UA即用户代理）

网站服务器通过识别 “UA”来确定用户所使用的操作系统版本、CPU 类型、浏览器版本等信息。

而网站服务器则通过判断 UA 来给客户端发送不同的页面。

所以可以用来进行身份证明
