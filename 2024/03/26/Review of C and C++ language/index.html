Review of C and C++ language

目录
  1. 输入和输出

在大三的这个学期，我又重新学习了数据结构这门课，发现自己当初学的很多东西都忘了

在学习了很多别的语言后，再来看大一的C语言，感觉又有了很多不一样的理解和体会

输入和输出

虽然输入和输出是C或C++的基础或者说是敲门砖，但是实际上很多人都没有彻底的搞清楚这个知识点

scanf()函数

scanf()函数默认是以空白字符（空格、制表符、换行符）为分隔

但是如果使用 %d %s 等格式制表符，scanf()就会忽略输入中的空白字符，并将它们作为分隔符

scanf()不会读取末尾的 ‘\n’ 换行符

Tips：这里要特别注意输入时，变量前的&符号什么时候要什么时候不要

printf()函数

printf()是C语言中标准的输出函数，因此要注意的就是格式化输出时的参数

常见的格式说明符有以下这些

getchar()函数

从标准输入流中读取一个字符，常用来处理上一行末尾留下的换行符

gets()函数

从标准输入流读取字符串，但是因为没有限制读入长度，所以是非常危险的函数，可以选择使用 fgets() 替代

gets()会把末尾的换行符一起读取了，但是之后会将其从输入中删除

puts()函数

只能用于输出字符串，无法使用格式控制符，并且末尾会自动输出一个换行符

fgetls()函数

用于从输入流中读取一整行的文本，并将其存储到指定字符串中

cin()函数

cin()是C++中的标准输入函数，完整的用法是 std::cin

但是我们平常使用的时候都会提前在开头使用 using namespace std;

这样就可以直接使用 cin 了

cout()函数

cout()是C++中的标准输出函数

sprintf()函数

sprintf()函数用于将格式化的数据写入到一个字符串中，而不是输出到标准输出流
