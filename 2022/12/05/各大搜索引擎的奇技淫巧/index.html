各大搜索引擎的奇技淫巧

每天都在频繁的用到各大搜索引擎，

但是却一直没有好好学习过如何高效的使用搜索引擎。

所以今天就来好好整理一下搜索的技巧吧（^_^）

Google

完全匹配搜索

那么如果我们想要完全匹配搜索的话，使用双引号：

比如在搜索框中输入“python”

模糊匹配搜索

使用*来代替你忘记的内容

比如在搜索框中输入py*on

排除搜索

如果你想在google中搜索的结果不要某些网站或者关键词出现，那么你可以用 - 来剔除

比如在搜索框中输入：肖申克的救赎 -维基百科

在特定的网站搜索

可以通过 site: 来指定我们想在特定的网站里面搜索相关的内容：

比如在搜索框中输入：python site:baidu.com

特定文件类型搜索

可以通过 filetype:来搜索文件

比如在搜索框中输入：身份证 filetype:xlsx

在网页标题, 链接和主体中搜索关键词

比如在搜索框中输入：intittle 后台登录

搜索相关网站

比如在搜索框中输入：related: baidu.com

用 OR (或)逻辑进行搜索

比如在搜索框中输入：”java” OR “python”

搜索的结果中会出现匹配 java 或 python 的结果。需要注意的是，OR一定要是大写。

用~来进行同义词搜索

比如在搜索框中输入：白蛇 ~电影

站内搜索

比如在输入框中输入：site:csdn.net java
